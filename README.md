Kanban Task Manager

A simple and interactive Kanban Task Manager Mobile Application built using Flutter.
This application helps users manage and organize their tasks visually using a Kanban Board system.

📌 Project Overview

The Kanban Task Manager allows users to manage tasks through different workflow stages using separate columns.

The board is divided into:

To Do
In Progress
Done

Users can efficiently track task progress and improve productivity through a clean and modern mobile interface.

✨ Features
Create and manage tasks
Organize tasks into Kanban columns
Simple and responsive UI
Interactive task cards
Edit and delete task functionality
Smooth mobile user experience
Easy task tracking system
🛠 Technologies Used
Flutter
Dart
🚀 How to Run the Project
Download or clone the repository
Open the project in Flutter-supported IDE
(VS Code or Android Studio)
Run the following command:
flutter pub get
Start the application using:
flutter run
